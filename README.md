# EdTech Learning Intelligence Platform

A modern AI-powered EdTech platform designed to help students, teachers, and institutions monitor learning progress, predict academic performance, and deliver personalized learning experiences.

## 📖 Overview

This project was designed as part of my **UI/UX Design Internship at Codec Technologies**.

The platform combines learning analytics, AI-driven insights, adaptive learning, and gamification to improve student engagement while maintaining productivity.

---

## 🎯 Project Objectives

- Track student learning behavior
- Visualize learning progress
- Predict academic performance using AI
- Recommend personalized learning paths
- Balance gamification with productive learning
- Improve engagement through intuitive UX
